# Getting Started

Welcome to Bklit Analytics! This guide will help you set up and start using Bklit in your project.

## What is Bklit?

Bklit Analytics is a modern, privacy-focused analytics platform that helps you understand your users without compromising their privacy. Built with Next.js, TypeScript, and a monorepo architecture, Bklit provides:

- **Real-time Analytics** - Track page views, sessions, and custom events
- **Privacy-First** - No cookies, GDPR compliant
- **Developer-Friendly** - Easy SDK integration, comprehensive API
- **Scalable** - Built on PostgreSQL and ClickHouse for performance
