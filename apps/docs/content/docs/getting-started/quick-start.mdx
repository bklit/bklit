Get Bklit Analytics running locally in minutes. This guide will help you set up the dashboard, SDK, and playground for development.

<Callout type="info">
  **Quick Setup:** To get started quickly, you only need the SDK, dashboard, and
  playground running. The website and Trigger.dev are optional for basic
  development.
</Callout>

## Prerequisites

- Node.js 22.14.0 or higher
- pnpm 9.6.0 or higher (recommended)
- PostgreSQL (<a href="https://console.prisma.io/sign-up" target="_blank">Prisma Postgres</a> recommended)
- GitHub OAuth app (for authentication)

## Installation

Clone the repository and install dependencies:

<Tabs items={["pnpm", "npm"]}>
  <Tab value="pnpm">

```bash
git clone https://github.com/yourusername/bklit.git
cd bklit
pnpm install
```

  </Tab>
  <Tab value="npm">

```bash
git clone https://github.com/yourusername/bklit.git
cd bklit
npm install
```

  </Tab>
</Tabs>

## Environment Setup

Create a `.env` file in the root directory with the following minimum configuration:

```bash
# Database
DATABASE_URL="postgresql://..."

# ClickHouse Analytics
CLICKHOUSE_HOST=
CLICKHOUSE_USERNAME=
CLICKHOUSE_PASSWORD=

# Authentication (BetterAuth)
AUTH_SECRET=
AUTH_GITHUB_ID=
AUTH_GITHUB_SECRET=

# Polar (Payments) - Development Mode
POLAR_SERVER_MODE="sandbox"
POLAR_ACCESS_TOKEN=
POLAR_WEBHOOK_SECRET=

# Monitoring
ALERT_EMAIL=
HEALTH_CHECK_SECRET=

# Mapbox (for location visualizations)
NEXT_PUBLIC_MAPBOX_TOKEN=
```

## Database Setup

Setup your postgres database, you can get one for free with <a href="https://console.prisma.io/sign-up" target="_blank">Prisma Postgres</a>. Generate Prisma client and run migrations:

<Tabs items={["pnpm", "npm"]}>
  <Tab value="pnpm">

```bash
pnpm db:generate
pnpm db:migrate
```

  </Tab>
  <Tab value="npm">

```bash
npm run db:generate
npm run db:migrate
```

  </Tab>
</Tabs>

## Start Development

Run the dashboard and playground:

<Tabs items={["pnpm", "npm"]}>
  <Tab value="pnpm">

```bash
pnpm dev
```

  </Tab>
  <Tab value="npm">

```bash
npm run dev
```

  </Tab>
</Tabs>

The dashboard will be available at `http://localhost:3000`
