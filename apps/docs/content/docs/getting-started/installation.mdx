# Installation

This guide will walk you through installing and setting up Bklit Analytics.

## Prerequisites

Before you begin, ensure you have:

- **Node.js 22.18.0** or higher
- **pnpm** (package manager)
- **PostgreSQL** database
- **ClickHouse** database (for analytics)

### Installing Node.js

If you need to install or switch Node versions, we recommend using [nvm](https://github.com/nvm-sh/nvm):

```bash
nvm install 22.18.0
nvm use 22.18.0
```

### Installing pnpm

```bash
npm install -g pnpm
```

## Project Structure

Bklit is a monorepo managed with [Turborepo](https://turbo.build/repo) and [pnpm workspaces](https://pnpm.io/workspaces).

```
bklit/
├── apps/
│   ├── dashboard/          # Main analytics dashboard (Next.js)
│   ├── playground/         # Demo app for SDK testing (Vite)
│   ├── website/            # Marketing website (Next.js)
│   └── docs/               # Documentation site (Fumadocs)
│
└── packages/
    ├── api/                # tRPC API routes
    ├── auth/               # Better Auth + Polar integration
    ├── db/                 # Prisma database layer
    ├── email/              # React Email templates
    ├── sdk/                # Analytics SDK (published)
    ├── ui/                 # Shared UI components (shadcn/ui)
    ├── utils/              # Common utilities
    └── validators/         # Zod schemas
```

## Installation Steps

### 1. Clone the Repository

```bash
git clone https://github.com/bklit/bklit.git
cd bklit
```

### 2. Install Dependencies

From the root directory:

```bash
pnpm install
```

This will install all dependencies for all apps and packages in the monorepo.

### 3. Set Up Environment Variables

Copy the `.env.example` file to `.env` in the root directory:

```bash
cp .env.example .env
```

Then fill in the required values. See the [Environment Variables](/docs/getting-started/environment-variables) guide for details.

### 4. Set Up the Database

Run Prisma migrations to set up your database schema:

```bash
pnpm db:migrate
```

Generate the Prisma client:

```bash
pnpm db:generate
```

### 5. Start the Development Server

From the root directory:

```bash
pnpm dev
```

This will start:
- Dashboard at `http://localhost:3000`
- Website at `http://localhost:3001`
- Docs at `http://localhost:3002`

Or run individual apps:

```bash
# Dashboard only
pnpm -F @bklit/dashboard dev

# Website only
pnpm -F @bklit/website dev

# Docs only
pnpm -F @bklit/docs dev
```

## Key Technologies

- **Framework:** Next.js 15 (App Router), React 19
- **Database:** PostgreSQL with Prisma ORM & ClickHouse for Analytics
- **Auth:** Better Auth with GitHub/Google OAuth
- **Billing:** Polar.sh
- **Email:** Resend with React Email
- **UI:** shadcn/ui with Tailwind CSS v4
- **API:** tRPC for type-safe APIs
- **Monorepo:** Turborepo + pnpm workspaces
