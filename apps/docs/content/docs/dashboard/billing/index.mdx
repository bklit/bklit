# Billing & Subscriptions

Bklit uses [Polar.sh](https://polar.sh) for billing and subscription management.

## Overview

Bklit offers two plans:

- **Free Plan** - Perfect for getting started
- **Pro Plan** - For growing teams

## Plans

### Free Plan

**Included:**
- 1 workspace
- 1 project per workspace
- 1 team member per workspace
- Real-time analytics
- Funnel analysis
- Custom event tracking
- API access
- Community support

**Limits:**
- 4,000 events per month
- Standard data retention

### Pro Plan

**Price:** $10/month

**Included:**
- Unlimited workspaces
- Unlimited projects
- Unlimited team members
- All analytics features
- Funnel analysis
- Priority support (Discord)
- Real-time analytics
- API access

**Limits:**
- 100,000 events per month (base)
- Usage-based billing: $0.0004 per additional event
- Extended data retention
