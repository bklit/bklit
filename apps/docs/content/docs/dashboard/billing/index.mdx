# Billing & Subscriptions

Bklit uses [Polar.sh](https://polar.sh) for billing and subscription management.

## Overview

Bklit offers two plans:

- **Free Plan** - Perfect for getting started
- **Pro Plan** - For growing teams

## Plans

### Free Plan

**Included:**
- 1 workspace
- 1 project per workspace
- 1 team member per workspace
- Basic analytics dashboard
- Page view tracking
- Community support

**Limits:**
- 10,000 events per month
- Standard data retention

### Pro Plan

**Included:**
- Unlimited workspaces
- 5 projects per workspace
- 5 team members per workspace
- Advanced analytics dashboard
- Custom event tracking
- Session recordings
- Priority support
- Extended data retention
- Custom domains
- API access
- Export capabilities

**Limits:**
- 100,000 events per month
- Extended data retention
