# Admin Features

Bklit includes admin features for managing accounts, bypassing limits, and whitelisting users.

## Overview

Admin features include:
- **Super Admin** - Unlimited access role
- **God Mode** - Bypass all limits
- **Whitelisting** - Grant special access
- **Account Management** - Manage user accounts
