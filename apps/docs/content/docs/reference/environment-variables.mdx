# Environment Variables Reference

Complete reference for all environment variables used in Bklit.

## Quick Reference

See [Environment Variables Guide](/docs/getting-started/environment-variables) for setup instructions.

## Variable Categories

### Database
- `DATABASE_URL` - PostgreSQL connection string

### Application URLs
- `AUTH_URL` - Dashboard URL for auth callbacks
- `NEXT_PUBLIC_APP_URL` - Client-side app URL

### Authentication
- `AUTH_SECRET` - Better Auth session secret
- `AUTH_GITHUB_ID` - GitHub OAuth client ID
- `AUTH_GITHUB_SECRET` - GitHub OAuth client secret
- `AUTH_GOOGLE_ID` - Google OAuth client ID (optional)
- `AUTH_GOOGLE_SECRET` - Google OAuth client secret (optional)

### Polar (Billing)
- `POLAR_ACCESS_TOKEN` - Polar API token
- `POLAR_SERVER_MODE` - `sandbox` or `production`
- `POLAR_WEBHOOK_SECRET` - Webhook verification secret
- `POLAR_ORGANIZATION_ID` - Polar organization ID
- `POLAR_PRO_PRODUCT_ID` - Pro plan product ID (server-side)
- `NEXT_PUBLIC_POLAR_PRO_PLAN_PRODUCT_ID` - Pro plan product ID (client-side)
- `POLAR_FREE_PRODUCT_ID` - Free plan product ID (optional)

### Analytics
- `CLICKHOUSE_HOST` - ClickHouse database host
- `CLICKHOUSE_USERNAME` - ClickHouse username
- `CLICKHOUSE_PASSWORD` - ClickHouse password

### Email
- `RESEND_API_KEY` - Resend API key

### Health Monitoring
- `ALERT_EMAIL` - Health check alert email
- `HEALTH_CHECK_SECRET` - Health check endpoint secret

### Trigger.dev
- `TRIGGER_SECRET_KEY` - Trigger.dev secret key (optional)
- `TRIGGER_API_KEY` - Trigger.dev API key (optional)
- `TRIGGER_API_URL` - Trigger.dev API URL (optional)

### Website App
- `NEXT_PUBLIC_BKLIT_WEBSITE_API_TOKEN` - Website tracking token
- `NEXT_PUBLIC_BKLIT_API_HOST` - API host override (optional)

### Dashboard App
- `NEXT_PUBLIC_MAPBOX_TOKEN` - Mapbox API token

### Optional
- `BKLIT_WEBSITE_URL` - Marketing website URL
- `BKLIT_DEFAULT_PROJECT` - Auto-invite project ID
- `NODE_ENV` - Node environment
