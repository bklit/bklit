# SDK Installation

This guide covers installing the Bklit Analytics SDK in your project.

## Package Managers

### npm

```bash
npm install @bklit/sdk
```

### pnpm

```bash
pnpm add @bklit/sdk
```

### yarn

```bash
yarn add @bklit/sdk
```

## Framework-Specific Guides

- [Vanilla JavaScript / React](/docs/sdk/vanilla-js) - Standard JavaScript setup
- [Next.js](/docs/sdk/nextjs) - Next.js App Router integration

## Requirements

- **Node.js:** 18+ (for build tools)
- **Browser:** Modern browsers (ES6+ support)
- **Project ID:** From your Bklit dashboard
- **API Key:** From your Bklit dashboard
