# Workspace Setup

A workspace (organization) in Bklit is a container for your projects and team members.

## Creating a Workspace

### During Onboarding

When you first sign up, you'll be prompted to create a workspace:

1. Enter your organization name
2. Click "Create" or "Next"
3. Workspace is created automatically

### After Onboarding

If you need to create another workspace:

1. Go to dashboard home
2. Click "Create Organization" or "+"
3. Enter organization name
4. Click "Create"

## Workspace Features

### Projects

- All projects belong to a workspace
- Projects are scoped to their workspace
- Multiple projects per workspace

### Team Members

- Invite team members to workspace
- Assign roles (owner, admin, member)
- Manage permissions

### Billing

- Billing is per workspace
- Plans apply to entire workspace
- Usage limits are workspace-wide

## Workspace Limits

### Free Plan

- 1 workspace per user
- 1 project per workspace
- 1 team member per workspace

### Pro Plan

- Unlimited workspaces
- 5 projects per workspace
- 5 team members per workspace

### Super Admin

- Unlimited workspaces
- Unlimited projects
- Unlimited team members

## Workspace Settings

Access workspace settings:

1. Go to workspace dashboard
2. Click "Settings" in sidebar
3. Manage workspace configuration

### Available Settings

- **General** - Name, logo, metadata
- **Members** - Team management
- **Billing** - Subscription and plans
- **API Tokens** - Token management
- **Projects** - Project list

## Workspace Roles

### Owner

- Full access to workspace
- Can delete workspace
- Manage billing
- Invite/remove members
- Create/delete projects

### Admin

- Manage projects
- Invite members (not owners)
- Manage API tokens
- Cannot delete workspace
- Cannot manage billing

### Member

- View projects
- View analytics
- Cannot manage settings
- Cannot invite members

## Best Practices

### Naming

- Use descriptive names
- Include company/team name
- Avoid generic names like "My Workspace"

### Organization

- One workspace per company/team
- Separate workspaces for different products
- Use projects to separate applications

### Team Management

- Add team members as needed
- Use appropriate roles
- Remove inactive members

## Troubleshooting

### Can't Create Workspace

- Check you're on Free plan (1 workspace limit)
- Verify you don't already have a workspace
- Check for account restrictions

### Workspace Not Found

- Verify you're in the correct account
- Check workspace wasn't deleted
- Contact support if needed
