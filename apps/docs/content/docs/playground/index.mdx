# Playground

The Bklit playground is a demo application for testing SDK integration and seeing analytics in action.

## What is the Playground?

The playground is a simple e-commerce demo app that:
- Demonstrates SDK integration
- Shows event tracking in action
- Allows testing without modifying your own app
- Provides a controlled testing environment

## Features

- **Product Catalog** - Browse products
- **Event Tracking** - Automatic and manual events
- **Real-time Analytics** - See events in dashboard
- **SDK Testing** - Test SDK configuration
