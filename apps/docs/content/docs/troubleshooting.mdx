# Troubleshooting

Common issues and their solutions.

## SDK Issues

### SDK Not Loading

**Symptoms:**
- No console logs
- Events not tracking
- SDK errors in console

**Solutions:**
- Check SDK is installed: `npm list @bklit/sdk`
- Verify initialization code
- Check browser console for errors
- Ensure project ID and API key are correct

### Events Not Appearing

**Symptoms:**
- Events sent but not in dashboard
- No data in analytics

**Solutions:**
- Verify API token is valid
- Check token has project access
- Verify project ID matches
- Check usage limits not exceeded
- Enable debug mode to see logs

### Debug Mode Not Working

**Symptoms:**
- No debug logs in console
- Can't see SDK activity

**Solutions:**
- Verify `debug: true` is set
- Check browser console is open
- Ensure not in production mode
- Check for console filters

## API Issues

### 401 Unauthorized

**Symptoms:**
- API returns 401 error
- "Invalid token" message

**Solutions:**
- Verify API token is correct
- Check token hasn't expired
- Ensure token format is correct
- Verify token has project access

### 403 Forbidden

**Symptoms:**
- API returns 403 error
- "Domain not allowed" message

**Solutions:**
- Check domain restrictions on token
- Verify request origin matches allowed domains
- Add domain to token settings
- Check for typos in domain

### 429 Too Many Requests

**Symptoms:**
- API returns 429 error
- "Usage limit exceeded" message

**Solutions:**
- Check current usage in dashboard
- Upgrade to Pro plan for higher limits
- Wait for monthly reset
- Optimize event tracking

## Database Issues

### Connection Errors

**Symptoms:**
- Database connection failed
- Prisma errors

**Solutions:**
- Verify `DATABASE_URL` is correct
- Check database is running
- Verify credentials
- Check network/firewall

### Migration Errors

**Symptoms:**
- Migrations fail
- Schema errors

**Solutions:**
- Check migration files
- Verify database state
- Run `pnpm db:generate`
- Check for conflicts

## Environment Variables

### Variables Not Loading

**Symptoms:**
- Environment variables not found
- Undefined values

**Solutions:**
- Check `.env` file exists
- Verify variable names
- Restart development server
- Check for typos

### Wrong Values

**Symptoms:**
- Using wrong environment
- Incorrect configuration

**Solutions:**
- Verify `.env` file location
- Check variable values
- Ensure correct environment
- Review configuration

## Billing Issues

### Subscription Not Updating

**Symptoms:**
- Plan not changing
- Features not enabled

**Solutions:**
- Check webhook configuration
- Verify Polar webhook secret
- Check webhook logs
- Verify subscription status

### Payment Failed

**Symptoms:**
- Payment error
- Subscription inactive

**Solutions:**
- Update payment method
- Check card details
- Contact Polar support
- Verify billing information
